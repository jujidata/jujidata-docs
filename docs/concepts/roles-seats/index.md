# 功能介绍：账号管理与席位

## 一、方案与定价

购买页：显示三种购买方案，费用、用途等信息

![rs-pricing](./images/rs-pricing.png)

### 免费版

- 发现并对接供给商与服务商
- 在平台创建、发布并管理标注项目
- 浏览超60种标注模板，查看标注内容与配置示例

赠送席位：

- 个人新用户注册后赠送一个月 VIP
- 企业认证之后赠送两个人月的席位

### 个人VIP

可以选择不同的支付方式：银行卡、支付宝、微信支付

**个人账号只能给自己购买 VIP**


选择 `购买时长` ，点击 `立即支付` 即可购买

![rs-person](./images/rs-person.png)


**团队的管理员及以上管理者，可以给团队所有成员购买 VIP**


1. 给自己购买 VIP，功能和操作与 `个人账号` 购买 VIP 完全相同

![rs-team1](./images/rs-team1.png)


2. 给指定的团队成员购买 VIP：选择至少一位团队成员，选择`购买时长` ，点击 `立即支付`即可购买

![rs-team2](./images/rs-team2.png)

**管理员可以给超级管理员购买 VIP**


### 企业VIP

给企业的员工购买和分配席位：选择购买时长（/人月），点击 `立即支付` 即可购买

![rs-enterprise](./images/rs-enterprise.png)


### 支付流程

点击 `立即支付` ，跳转到支付宝付款页面

![rs-pay1](./images/rs-pay1.png)

扫码之后就会出现下面的页面

![rs-pay2](./images/rs-pay2.png)

二维码每隔一分钟刷新一次，若扫码后支付失败不需要重新扫码，只需要重新输入密码即可

此订单 60 分钟内有效

完成付款之后，跳转至以下页面，并显示订单号、订单金额

![rs-result1](./images/rs-result1.png)

稍等一会，就变成以下页面：

![rs-result2](./images/rs-result2.png)

点击上面这个页面的 `同步支付状态` 按钮，就会出现以下页面：

![rs-result3](./images/rs-result3.png)

点击 `订单号` 可以跳转至订单详情页面如下图所示：

![rs-billing](./images/rs-billing.png)

点击 `返回首页` 按钮，则跳转至首页


## 二、分配席位

### 企业席位

此页面上方的卡片分别显示剩余额度、已有席位的成员、可分配成员

![rs-seats](./images/rs-seats.png)

**企业超级管理员和管理员可以给该企业的任何一个成员分配、追加、撤回、回收席位**

1. 追加席位：可追加不大于 `额度余额` 的席位数量

![rs-extend](./images/rs-extend.png)

2. 撤回席位：能撤回该成员所有的全部的席位

![rs-held](./images/rs-held.png)

其中，只能回收本月之后的席位，即当前月的席位一经分配不可回收

### 席位标识

当账号具备有效席位时，侧边栏会显示皇冠标识，表示该账号当前可使用需要席位的功能。常见场景如下：

- 个人账号购买 `个人VIP` 并生效后
- 企业账号购买 `企业VIP`，且席位已分配到该账号后

![rs-queen](./images/rs-queen.png)
