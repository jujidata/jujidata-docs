## 3. 多元的平台服务模式

* **全包模式**：客户只提供数据，平台全包（从发布需求 -> 智能竞标/试标 -> 线上签约托管 -> 执行标注 -> 验收下载）。

![交易流程](/images/workflow.png)
* **自标模式**：客户提供数据 + 标注团队，平台仅提供工具（按需开通自标账号 -> 自主标注）。

![自标模式](/images/self_annotation.png)

## 4. 可靠的质量管控能力

* **全流程可视化**：甲方管理页面实时追踪任务进度，早发现、早处理。
* **严苛质检回溯（打回机制）**：甲方乙方均可在线审核，不合格数据一键打回。
* **重标留痕**：所有打回重标的操作、修改前后的对比数据均记录在案，作为最终结算的质量依据。

![标注管理版](/images/labeling_dashboard.png)
