# 🏠 自标注模式上手指南

如果您拥有自己的标注团队，并希望利用 **据吉网** 专业、圆润易用的 UI 工具进行内部协作，请选择“自标注”模式。该模式支持您自主管理成员、分配任务并直接获取成果。

---

## 🟢 第一阶段：项目创建与配置

### 1. 发布自标注项目
* **操作路径**：进入 `项目管理` 页面 -> 点击 `发布项目`。
* **关键选择**：在发布选项中，找到 **“标注模式”**，勾选 **[自标注]** 选项。
* **配置信息**：填入项目名称和任务类型（如：图片分割、语义理解），保存并生成项目卡片。

### 2. 开启组织成员管理
* **进入管理**：在项目列表中找到刚才创建的项目卡片。
* **管理入口**：点击卡片下方的 **[组织成员管理]** 图标。在这里，您可以邀请您的团队成员加入特定项目。

---

## 🟡 第二阶段：资源准备与团队建设

### 3. 为标注人员购买席位
为了保障平台工具的高效运行，自标注模式需要为每位参与的标注人员分配“席位”。
* **购买操作**：在成员管理界面，点击 **[购买/增加席位]**。
* **席位分配**：根据您的团队规模选择购买数量。支付成功后，即可将席位授权给指定的团队成员。

### 4. 上传项目原始数据
* **操作路径**：返回 `项目管理` -> 进入具体项目的 **[数据中心]**。
* **批量上传**：将您的原始素材（图片、音频或视频）打包上传。
* **任务预处理**：系统会自动对数据进行分包，以便后续分配给不同的标注席位。
